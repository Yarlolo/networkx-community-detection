# Сравнение методов выделения сообществ в графах
### Автор: Лосев Ярослав
---

## О проекте

Данный репозиторий содержит код и анализ для курсовой работы по теме **"Сравнение различных методов выделения сообществ на крупных графовых наборах данных"**.

**Цель исследования** — сравнить алгоритмы поиска сообществ (кластеров) в графах по качеству разбиения и вычислительной эффективности. Работа включает в себя как реализацию алгоритмов "с нуля" для глубокого понимания их принципов, так и их тестирование на реальном графовых датасетах.

---

**Основные результаты**:
*   Проведен обзор и классификация методов (дивизивные, оптимизации модулярности, эвристические, спектральные, на основе случайных блужданий, машинного обучения).
*   Реализованы ключевые алгоритмы: `Girvan-Newman`, `Greedy Modularity (CNM)`, `Louvain`, `Label Propagation` и другие.
*   Показано, что **алгоритм распространения меток (LPA)** является предпочтительным выбором для больших графов благодаря высокой скорости и хорошему качеству, в то время как **Girvan-Newman** и **Greedy Modularity** дают высокое качество, но не масштабируются.

---

## Основные выводы исследования

| Метод | Семейство | Качество (Q) | Время (сек) | Масштабируемость |
| :--- | :--- | :--- | :--- | :--- |
| **Girvan-Newman** | Дивизивные | Высокое | Очень высокое |  Плохая |
| **Greedy Modularity** | Оптимизация модулярности | Высокое | Очень высокое |  Плохая |
| **Louvain** | Оптимизация модулярности | Среднее | Среднее |  Хорошая |
| **Label Propagation (LPA)** | Эвристические | Среднее | Низкое |  **Отличная** |
| **Spectral Clustering** | Спектральные | - | - |  Плохая |
| **Walktrap** | Случайные блуждания | - | - |  Плохая |

**Ключевой вывод:** На рассмотренном датасете **алгоритм распространения меток (LPA)** является наиболее сбалансированным и предпочтительным выбором
